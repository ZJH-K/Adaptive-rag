# Built-in Knowledge Documents

The five files under `knowledge/markdown/` and `knowledge/pdf/` are small,
project-authored technical fixtures for Adaptive RAG development and acceptance
testing. They contain no copied third-party prose, credentials, or personal data.

## Documents

- `markdown/langgraph_checkpoint.md`: checkpoint persistence and `thread_id`.
- `markdown/embedding_batching.md`: document/query embedding and failure rules.
- `markdown/context_citation_guide.md`: context budgets and citation metadata.
- `pdf/dense_retrieval_guide.pdf`: two-page dense retrieval and persistence guide.
- `pdf/ingestion_recovery_manual.pdf`: three-page ingestion recovery workflow.

These fixtures are distributed under the same license terms as this repository.
The PDFs are born-digital documents with extractable text; OCR is not required.
